## Usage

Download

```cmd
git clone https://github.com/janselroa/pricing-component.git
```

```cmd
yarn install
```

```cmd
yarn webpack serve
```

open <a href="http://localhost:3000">localhost:3000</a>

### npm

```cmd
npm install
```

```cmd
npx webpack serve
```
